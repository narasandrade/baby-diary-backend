# Baby Diary - Backend

This is the backend for **Baby Diary**, developed with **Node.js + TypeScript + Express + MongoDB**.  
Responsible for providing the REST API to the frontend.

---

## 🚀 Technologies
- Node.js
- TypeScript
- Express
- MongoDB + Mongoose
- Dotenv

---

## 📦 Installation and Usage

### 1. Clone the repository
```sh
git clone https://github.com/narasandrade/baby-diary-backend.git
cd baby-diary-backend
```
### 2. Install dependencies
```sh
npm install
```
### 3. Configure environment variables
Create a .env file in the project root based on .env.example:
```sh
MONGO_URI=your_mongodb_connection_string
PORT=5000
```
### 4. Run the application
```sh
npm run dev
```
